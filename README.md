## Install GLNS package
<a href="https://ece.uwaterloo.ca/~sl2smith/GLNS/">GLNS Documentation</a>
1. Start Julia from command-line
```
$ julia
```
2. Import Pkg
```
julia> import Pkg
```
3. Install GLNS Package for Julia
```
Pkg.add('GLNS')
```
